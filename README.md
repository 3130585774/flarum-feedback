A [Flarum](https://flarum.org) extension. 主页反馈入口

## Installation

Install with composer:

```sh
composer require 3130585774/flarum-feedback:"*"
```

## Updating

```sh
composer update 3130585774/flarum-feedback:"*"
php flarum migrate
php flarum cache:clear
```
